# mini-projets-python

Ce dépôt contient plusieurs mini-projets réalisés en Python dans le cadre de mon apprentissage en programmation.

## Projets disponibles.

### 1. Hello World
**Fichier** : HelloWorld.py
\nCe programme est mon tout premier programme permettant d'afficher un message de bienvenue !

### 2. Etat de l'eau
**Fichier** : EtatDeLeau.py
\nCe programme permet de determiner l'état d'une eau à travers sa temperature entrée par l'utilisateur :
  - Solide
  - Liquide
  - Gaz

### 3. Mention au Bac
**Fichier** : MentionAuBac.py
\nCe programme affiche les différentes mentions qu'on peut avoir au Bac en fonction des différentes notes saisies par l'utilisateur !

### 4. PGCD ET PPCM
**Fichier** : PGCD ET PPCM.py
\nCe programme m'a permis de voir l'importance des fonctions en python et de les utiliser à calculer des pgcd et aussi ppcm !

### 5. Table de Multiplication
**Fichier** : TableDeMultiplication.py
\nCe programme demande a l'utilisateur d'entrer un nombre dont la machine devra afficher la table de multiplication !

### Technologies utilisées
  - Python 3

### Objectifs
\nCes mini projets ont été réalisés dans le cadre de ma formation académique mais aussi dans le but de renforcer mes compétences en :
- Algorithmiques ;
- Boucles ;
- Fonctions ;
- Manipulation des entrées et sorties ;
- Structures conditionnelles.
